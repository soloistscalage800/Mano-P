## 🚀 What Mano-P does

Mano-P is a local GUI-VLA agent that helps you automate tasks on your computer by looking at the screen and acting like a person would. It can open apps, click buttons, fill forms, and move through multi-step tasks.

It is built for edge devices and local use. That means your data stays on your device during use.

## 🪟 System requirements

Mano-P is designed for local use on modern Windows PCs. For the best result, use a machine with:

- Windows 10 or Windows 11
- 8 GB RAM or more
- A recent Intel or AMD CPU
- A stable internet connection for the download
- A screen resolution of 1366 × 768 or higher

If your computer has a faster GPU or an NPU, Mano-P can run more smoothly, but it can still work on a normal laptop.

## ✍️ How to use Mano-P

Use plain language. For example:

- Open Chrome and search for train times
- Copy the latest invoice into Excel
- Open Settings and change the display brightness

Mano-P plans the steps, watches the screen, and carries out the task. For longer jobs, it can handle one step at a time until the task is done.

## 🧩 Tips for better results

Use short, direct instructions.

- Say one task at a time
- Keep app names clear
- Use exact file names when possible
- Let the app finish each step before adding more
- Make sure the window you want is visible on screen

If the task needs a login, open the app or browser first so Mano-P can work from the right place.

## ❓ Common questions

### Do I need coding skills?
No. Mano-P is meant for regular computer users.

### Does it work offline?
The core idea is local use. Once the app is downloaded, it can run tasks on your device without sending your data away.

### Can it control any app?
It works best with common desktop apps and websites that have clear buttons and text fields.

### What if it clicks the wrong thing?
Use simpler instructions and keep the target window in view. You can stop the task and try again with a clearer prompt.
